# Instructions:

Simply run the project in through Visual Studio
(release mode recommended)

## Application hotkeys:
C - Unlock camera/mouse

W - Move camera forwards
A - Move camera Left
S - Move camera backwards
D - Move camera right

Space - Move camera up
Left Ctrl - Move camera down

Q - Rotate camera left
E - Rotate camera right

Esc - Exit application

## Use ImGui overlay to see:
- Resources currently stored in ResourceManager
- All used allocators along with their individual allocations and memory usage visualizations
